# Diretório de Imagens dos Quartos

Este diretório armazena as imagens dos quartos de cada navio.

## Estrutura:
```
fotos-quartos/
├── {id_quarto_1}/
│   ├── imagem1.jpg
│   ├── imagem2.png
│   └── ...
├── {id_quarto_2}/
│   ├── imagem1.jpg
│   └── ...
└── ...
```

## Formato das imagens:
- Formatos aceitos: JPG, PNG, GIF, WebP
- Tamanho máximo: 10MB por arquivo
- Resolução recomendada: até 1920x1080

## Observações:
- As imagens são organizadas por ID do quarto
- Cada quarto pode ter múltiplas imagens
- Os nomes dos arquivos são gerados automaticamente com UUID para evitar conflitos
